# Volumetric Modeling
## Voxel Modeling
Unity project for generating voxel-based shapes.

## Description
This directory contains the source code of a Unity project in which I developed scripts for generating voxel-based spheres.
The script allows you to configure the number of spheres, the voxel size, and the logical behavior between spheres (Union, Intersection, Complement), as well as to simulate a destruction tool to observe the implementation of a scalar value contained in each voxel defining its visibility.
