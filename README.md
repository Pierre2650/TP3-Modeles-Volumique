# Modélisation Géométrique - GAMAGORA
## Modélisation de Maillages
Projet Unity pour la génération de maillages de différentes figures géométriques.

## Description
Le repertoire contient le code source et le build d’un projet Unity dans lequel j’ai développé des scripts permettant de générer les maillages d’un plan, d’un cylindre, d’une sphère, d’un cône et d’une sphère tronquée, chacun avec différents paramètres configurables.
Le build permet d’observer le bon fonctionnement des scripts en cliquant sur des boutons pour générer les différents maillages.
